ANIMALITY TYPING MECHANICS

1. CORE FUNCTION

The system describes how instinctive responses form under perceived instability. It models transformation from internal processing into external behavioral expression.

A type is a structured chain consisting of Inner, Outer, and optional Influence applied to Outer.

2. PROCESS FLOW

Internal processing always follows this sequence:

INNER → OUTER → INFLUENCE (applied to OUTER)

Inner determines instinct direction. Outer determines behavioral expression. Influence modifies how Outer is expressed without changing Inner structure.

3. INNER FUNCTION

Inner represents the core instinctive processing response activated by instability.

It defines the underlying drive, need, or pressure resolution direction.

Inner does not appear directly in behavior. It determines the formation of Outer.

4. OUTER FUNCTION

Outer is the external behavioral output generated from Inner processing.

It represents observable reaction in action, communication, or environmental response.

Outer is shaped by internal processing and adaptation patterns.

5. INFLUENCE FUNCTION

Influence is a non-core modifier applied to Outer expression only.

It does not alter Inner and does not generate behavior independently.

Influence affects expression characteristics such as intensity, framing, selection of detail, pacing, or behavioral emphasis depending on the type of influence present.

Influence is optional. If absent, Outer expression remains unmodified.

6. LIMINAL STATE

Liminal occurs when Inner and Outer are identical.

This means no transformation gap exists between internal processing and external expression.

INNER = OUTER

Behavior is expressed directly from internal processing without reinterpretation between layers.

Influence may still exist in liminal structure but remains optional and does not affect whether a type is liminal.

7. WRITTEN FORMAT

Types are written using a fixed syntax structure.

Standard form:

<ex>OUTER >in<INNER{INFLUENCE}

Rules:

- <ex> = Outer Animal (external expression)
- >in< = Inner Animal (internal processing source)
- {Influence} = optional modifier applied to Outer expression

If Influence is not present:

- The { } section is completely omitted
- No placeholder is used

8. LIMINAL WRITTEN FORMAT

When Inner and Outer are identical, the format becomes:

<lm<ANIMAL{INFLUENCE}

Rules:

- <lm< replaces <ex> and >in<
- No Inner field is written separately
- Influence remains optional and follows standard rules
